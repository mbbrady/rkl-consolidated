## About This Repository

This is an internal review version of the RKL website built with [Hugo](https://gohugo.io/). The site is password-protected when deployed and is not intended for public access during the formation and ethics review period.

## Build Instructions

```bash
hugo server
```

For production build:
```bash
hugo --gc --minify
```

## Deployment

This site is configured for deployment to Netlify with password protection enabled for internal review.
